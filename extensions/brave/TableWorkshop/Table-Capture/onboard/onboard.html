Basic Mechanics

There are two ways to invoke the extension.

The Workshop is the panel that appears at the bottom (or top if you choose to dock it there) of the page that houses all of the newest Table Capture features.
The Popup Window is what appears when you click the extension icon in the toolbar. It's best used for exporting multiple tables on the page simultaneously.
The Table Capture Workshop

The Workshop is the preferred way to interact with the extension.

Highlight any text within the table you're interested in and control clickright-click it. Select "Table Capture - Launch workshop for selection". This will launch the Workshop with the context it needs to know which table you care about.
The Table Capture Popup Window
Try opening the Popup Window, selecting all of the tables on this page and exporting them to Google Sheets in one batch.
You can find the icon for the Popup Window in your browser's toolbar or extension tray.
