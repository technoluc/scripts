Keyboard Shortcuts
You can customize the keyboard shortcut that opens the popup window here: chrome://extensions/shortcuts
Local HTML Files
You can use the extension on local files, but first you'll need to navigate to the Chrome Extension management page (chrome://extensions), click the "Details" button for Table Capture and then enable the "Allow access to file URLs" toggle.

Frequently Asked Questions

  • Why does my data have a lot of links in it?

    You can disable link (web address) extraction in the options page.

  • Can I copy multiple tables at the same time?

    Yes you can! From the Table Capture popup window, select the tables you'd like to copy and then use the corresponding action buttons.

  • I can't see my table - what do I do?

    Try highlighting some text in your table, right-clicking and opening the Table Capture Workshop.

  • Are you available for 1:1 training sessions?

    Yup! Just email me and we can schedule a session where I can walk you through the extension, help you get the most from it.
